# **App Name**: CodeDeploy

## Core Features:

- Code Editor: A text editor interface where users can write and edit HTML, CSS, and JavaScript code.
- Live Preview: A real-time preview of the code output as it would appear in a browser.
- Deployment Automation: Automatically commit and push code changes to the specified GitHub repository (adbossappmaker/sites) using the provided GitHub personal access token for deployment to GitHub Pages. LLM Tool will use the access token for making decisions of when and how to make deployment.
- GitHub Authentication: Securely authenticate users using their GitHub credentials to manage deployment permissions.
- Dependency Suggestion: Suggest useful HTML, Javascript, and CSS libraries based on user code patterns.
- Error Checking and Suggestions: Show a list of possible compile-time and/or runtime errors.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to reflect stability and technological focus.
- Background color: Light gray (#F0F2F5), nearly desaturated and bright to create a neutral, clean workspace.
- Accent color: Purple (#6639A6), providing a modern and contrasting highlight.
- Body and headline font: 'Inter', a sans-serif font for a modern and readable interface.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, minimalist icons for actions like save, deploy, and preview.
- Subtle transitions and animations to provide feedback on actions like code deployment.
